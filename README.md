
# 📝 To-Do List App

A simple and responsive **To-Do List Web App** built using **HTML**, **CSS**, and **JavaScript**.

## 🚀 Features

- ✅ Add tasks with:
  - Task Name
  - Deadline
  - Priority (High / Medium / Low)
- 📅 Tasks are saved in **LocalStorage**
- 🗑️ Delete tasks anytime
- 🌈 Stylish and responsive user interface

## 💻 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage API
